# Login Shell Spec

## Contrato visual comun

- Base clara y premium, sin temas oscuros ni neones.
- Contenedor centrado con max-width comodo para formularios.
- Radio amplio (`24px` a `32px`).
- Borde claro y sombra premium.
- Inputs suaves, consistentes y accesibles.
- CTA propio por app, pero con las mismas proporciones.

## Capas

1. Fondo atmosferico.
2. Panel glass.
3. Bloque de marca.
4. Titulo/contexto.
5. Formulario o contenido auth.
