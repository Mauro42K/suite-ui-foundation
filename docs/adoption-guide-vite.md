# Adopcion en Vite + React

1. Copiar `react/` a `src/components/suite-auth/`.
2. Reutilizar el mark existente de la app y pasarlo como `mark`.
3. Sustituir la pantalla de login por `SuiteAuthShell`.
4. Mantener handlers, i18n y wiring de auth intactos.
5. Validar con `npm run typecheck` y `npm run build`.
