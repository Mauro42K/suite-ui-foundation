# suite-ui-foundation

Base visual compartida para la suite Mauro42K en la primera etapa de estandarizacion de experiencias auth/login.

## Alcance actual

- Tokens visuales de login
- Fondo ambiental de suite
- Panel premium tipo glass
- Lockup de marca
- Shell auth reutilizable para React
- Guias de adopcion para Next.js App Router y Vite + React

## Consumo en Fase 1

Esta base no se publica como paquete npm. Las apps consumidoras copian los archivos necesarios desde este repo y los mantienen como derivados controlados.

## Fuente visual

El baseline actual se deriva de `portal-web`, que define el fondo atmosferico y el panel premium aprobados para la suite.
