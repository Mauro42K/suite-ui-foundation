# App Icon Guidelines

## Reglas

1. Cada app mantiene su identidad propia.
2. Todos los marks deben sentirse parte de la misma suite.
3. Los marks deben vivir bien dentro de un frame glass redondeado.
4. Deben seguir viendose claros en 24px, 28px y 44px.
5. Si usan gradients SVG, deben evitar colisiones de `id` al renderizar multiples instancias.

## Recomendacion

- Preferir SVG como componente React para UI interna.
- Mantener un archivo SVG puro adicional para favicon o assets estaticos si hace falta.
